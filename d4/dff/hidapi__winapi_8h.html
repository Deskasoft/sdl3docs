int hid_winapi_descriptor_reconstruct_pp_data ( void * hidp_preparsed_data,
unsigned char * buf,
size_t buf_size )

Reconstructs a HID Report Descriptor from a Win32 HIDP_PREPARSED_DATA structure. This reconstructed report descriptor is logical identical to the real report descriptor, but not byte wise identical.

Parameters
[in]hidp_preparsed_dataPointer to the HIDP_PREPARSED_DATA to read, i.e.: the value of PHIDP_PREPARSED_DATA, as returned by HidD_GetPreparsedData WinAPI function.
bufPointer to the buffer where the report descriptor should be stored.
[in]buf_sizeSize of the buffer. The recommended size for the buffer is HID_API_MAX_REPORT_DESCRIPTOR_SIZE bytes.
Returns
Returns size of reconstructed report descriptor if successful, -1 for error. < API export macro < API call macro < API export and call macro
