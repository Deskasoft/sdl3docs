Convert URI to a local filename, stripping the "file://" preamble and hostname if present, and writes the result to the dst buffer. Since URI-encoded characters take three times the space of normal characters, src and dst can safely point to the same buffer for in situ conversion.

Returns the number of decoded bytes that wound up in the destination buffer, excluding the terminating NULL byte.

On error, -1 is returned.
